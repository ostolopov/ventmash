О проекте

Учебный MVP: каталог и карточки промышленных вентиляторов, источник данных — CSV. Бэкенд на Node.js + Express, фронтенд — Vanilla JS + Bootstrap 5.

Контакты

Для демонстрации в университете. Можно расширить: корзина, заявки, личный кабинет, админка.
