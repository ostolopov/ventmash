# VENTMASH — каталог промышленных вентиляторов

Учебный MVP B2B-каталога: Python (Flask), CSV → планируется БД, фронтенд на Bootstrap 5, стиль в духе Яндекса.

## Быстрый старт

```bash
./run.sh
```

Сервер: http://localhost:5000
